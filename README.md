# Dr. Hakyll

```
$ git clone https://github.com/stackbuilders/dr-hakyll
```

```
$ cabal sandbox init
$ cabal install --only-dependencies
$ cabal configure
$ cabal build
$ cabal run build
$ cabal run watch
```

```
$ stack setup
$ stack build
$ stack exec dr-hakyll build
$ stack exec dr-hakyll watch
```

```
$ stack exec dr-hakyll clean
$ stack exec dr-hakyll build
```

```
$ stack exec dr-hakyll rebuild
```
